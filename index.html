Nhánh main

  1. Xem nhánh local: git branch.
  2. Xem nhánh cả local và remote: git branch -a.
  3. Copy toàn bộ code ở nhánh hiện tại sang nhánh mới, con trỏ vẫn ở nhánh hiện tại: git branch -c tên_nhánh_mới.
  4. Copy toàn bộ code ở nhánh hiện tại sang nhánh mới, con trỏ sang nhánh mới: git checkout -b tên_nhánh_mới.
  5. Để chuyển con trỏ sang 1 nhánh nào đó: git checkout tên_nhánh.
  6. Push code từ nhánh con lên github: git push -u origin tên_nhánh.(từ lần 2 chỉ cần git push)
  7. Xóa nhánh trên máy tính(local repo): git branch -d tên_nhánh.(lưu ý không xóa được nhánh đang đứng và nhánh vừa commit nhưng chưa push lên github)
  8. Nếu nhánh có code chưa push sẽ ép xóa bằng: git branch -D tên_nhánh.
  9. Xóa nhánh trên remote repo: git push -d origin tên_nhánh.
